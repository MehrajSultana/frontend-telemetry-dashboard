# Frontend Telemetry & Observability

A frontend monitoring dashboard built with HTML, CSS, and JavaScript.

## Features
- KPI cards for active users, page views, error rate, load time, feature adoption, and API latency
- Usage trend chart section
- Dynamic frontend event log table
- Severity filters for info, warning, and error events
- CSV export for event logs
- Refresh button for simulated metric updates
- Responsive dashboard layout

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
This project demonstrates frontend telemetry tracking, observability metrics, usage monitoring, and event reporting for portfolio use.
